# コミットコマンド

現在の変更をコミットし、リモートにプッシュする。

## 使用方法
```
/commit
```

## 手順
1. `git status` で変更を確認
2. Conventional Commits形式でコミットメッセージ作成
3. `git push` でリモートに反映

## Conventional Commits形式
- `type(scope): subject`
- タイトルは50文字以内、本文は72文字で改行
- 動詞は原形（add, fix, update等）
- scopeは原則記述、適切なものがなければ...
