# dotfiles

## Usage

```bash
mkdir -p ~/src
cd ~/src
git clone https://github.com/Doarakko/dotfiles.git
cd dotfiles
```

### Claude Code

#### Plugin

```bash
/plugin marketplace add ~/src/dotfiles
/plugin install doarakko-config@dotfiles
```
